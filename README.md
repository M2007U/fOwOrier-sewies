# ‚ú® Features
*these features are available on our Website, partially available on GitHubPage*
- A huge canvas for the rendered result and the looping animation
- Settings are categorized into 4 pages :
  - Harmonics : this is where the user can set the parameters of the spinning vectors
  - Misc : contains settings unable to be categorized
  - Physics : defines how trails moves *(there are 3 types of trails : Tracing Trail, Coordinate Trail, and Wiping Trail)*
  - Visuals : let's the user define the colors of the graph *(most WebApps don't let you do this)*
- 12 presets
- 1 Article to explain Fourier Series better than this READ ME file
